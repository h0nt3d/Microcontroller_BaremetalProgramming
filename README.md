# Microcontroller_BaremetalProgramming
A collection of projects involving direct memory mapping to GPIO registers development boards (Arduino UNO / Freenove Board, ESP32, etc)

# Packages / Dependencies
### UNO / Freenove Control Board
- [avr-gcc](https://avrdudes.github.io/avr-libc/)
- [avrdude](https://github.com/avrdudes/avrdude)

### ESP32
- [esp-idf](https://github.com/espressif/esp-idf)
